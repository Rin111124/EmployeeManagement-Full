# PlantUML Sources

Bộ file PlantUML này được chia thành 2 phần mềm chính:

- `admin-system/`: phân tích và thiết kế cho Admin System
- `attendance-system/`: phân tích và thiết kế cho Attendance System

Các sơ đồ dùng chung như yêu cầu, class, ER và deployment vẫn nằm ở cấp gốc của thư mục `puml/`.
